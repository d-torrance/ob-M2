#+TITLE: Org Babel support for Macaulay2

This package adds support for working with Macaulay2 source code in Org Mode.

: #+BEGIN_SRC M2
:  R = QQ[x, y, z, w]
:  monomialCurveIdeal(R, {1, 2, 3})
: #+END_SRC

: : ideal (z  - y*w, y*z - x*w, y  - x*z)
